## DESCRIPCIÓN / CONTEXTO DEL PROYECTO
mesapass-api es una API REST desarrollada en Python con FastAPI que servirá como backend para la gestión de restaurantes, usuarios e interacciones dentro del sistema MesaPass.

El proyecto está diseñado siguiendo buenas prácticas de arquitectura backend (separación en capas, uso de ORM, migraciones) y se encuentra actualmente en fase de preparación técnica (Fase 0), donde se establece la base del entorno antes de implementar la lógica de negocio.

## Tecnologías utilizadas

Python 3.12
FastAPI
Uvicorn
PostgreSQL
SQLAlchemy 2.0
Alembic
Git & GitHub
